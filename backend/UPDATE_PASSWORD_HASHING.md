# Обновление системы хеширования паролей

## Изменения

Удален `bcrypt` и заменен на `passlib` с алгоритмом `pbkdf2_sha256`.

### Преимущества pbkdf2_sha256:

- ✅ Не требует дополнительных зависимостей (встроен в passlib)
- ✅ Совместим со всеми версиями Python
- ✅ Безопасный алгоритм (используется в Django по умолчанию)
- ✅ Нет ограничений на длину пароля
- ✅ Работает стабильно без проблем совместимости

## Установка

```bash
cd backend
venv\Scripts\activate  # Windows
pip uninstall bcrypt
pip install passlib==1.7.4
```

Или просто обновите зависимости:
```bash
pip install -r requirements.txt
```

## Проверка

После обновления попробуйте снова создать администратора:

```bash
python create_admin_simple.py
```

Теперь должно работать без ошибок!

## Миграция существующих паролей

Если у вас уже есть пользователи с паролями, захешированными через bcrypt, их нужно будет пересоздать или обновить пароли.
